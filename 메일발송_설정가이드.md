## ❓ 자주 묻는 질문

### Q1. 메일이 안 와요!
**A.** 다음을 차례로 확인하세요:
1. **꼭 https:// 주소로 접속**해서 신청했는지 확인 (file:// 로 열면 작동 안 함)
2. **스팸함(정크메일함)** 을 꼭 확인 — 첫 메일은 스팸 분류 빈번
3. **활성화 메일을 아직 클릭하지 않았다면** 메일은 보류 상태입니다 — 3단계 진행
4. ⏱ 1~2분 정도 기다려 보세요
5. 그래도 안 오면 신청서를 한 번 더 제출

### Q2. 자동 답장이 영어 제목으로 와요. 한글로 바꿀 수 없나요?
**A.** FormSubmit.co 무료 플랜은 자동 답장의 **본문은 한글**로 보내지만 **제목은 고정 영문**입니다.
제목까지 완전히 한글로 바꾸려면 FormSubmit Pro(유료) 또는 다른 서비스가 필요합니다.
지금은 본문 첫 줄에 신청자 이름과 "센텀Q AI 연구소"가 들어가므로 충분히 알아볼 수 있습니다.

### Q3. 한 번 활성화하면 영원히 유효한가요?
**A.** 네. **이메일 주소(phjcom3@gmail.com)** 단위로 한 번만 활성화하면 됩니다.
페이지를 다른 도메인으로 옮기거나 코드를 수정해도 다시 활성화할 필요가 없습니다.

### Q4. 메일이 갑자기 안 오기 시작했어요.
**A.** FormSubmit 무료 플랜은 **월 50건** 까지입니다 (이메일 주소당).
한 달에 50건을 넘으면 일시 중단되고, **다음 달 1일에 자동으로 초기화** 됩니다.

### Q5. 신청 내용을 받는 이메일을 추가하고 싶어요.
**A.** `index.html` 파일의 다음 줄을 찾아서 수정 (Ctrl+F):
```javascript
const ADMIN_EMAIL = 'phjcom3@gmail.com';
```
새 이메일로 바꿀 경우, 그 새 이메일에서도 **3단계 활성화 메일 클릭**을 한 번 해야 합니다.

### Q6. 키 같은 거 따로 입력 안 해도 되나요?
**A.** 네! FormSubmit.co는 **API 키가 없습니다.** 이메일 주소만 있으면 됩니다.
이미 코드에 `phjcom3@gmail.com` 이 들어가 있으므로 추가 작업이 필요 없습니다.

## 🔧 기술 정보 (참고용)

- **사용 서비스:** [FormSubmit.co](https://formsubmit.co) (무료, 가입 불필요)
- **엔드포인트:** `https://formsubmit.co/ajax/phjcom3@gmail.com`
- **전송 방식:** ⭐ **JSON POST** (`Content-Type: application/json`)
- **자동 답장:** `_autoresponse` 필드 (무료 제공)
- **메일 디자인:** `_template: "table"` (표 형식)
- **캡차:** `_captcha: "false"` (비활성화)

> ⚠️ 이전에 사용하던 **FormData(multipart) 방식은 일부 환경에서 차단**되어
> JSON 방식으로 변경했습니다. blogmaster 사이트와 100% 동일한 패턴입니다.
